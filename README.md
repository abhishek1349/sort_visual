# Sorting Algorithm Visualizer

A simple and beginner-friendly Sorting Algorithm Visualizer built using HTML, CSS, and JavaScript.

## Features

* Generate random arrays
* Visualize sorting step-by-step
* Adjustable speed control
* Adjustable array size
* Simple and clean UI
* Beginner-friendly code structure

## Sorting Algorithms Included

* Bubble Sort
* Selection Sort
* Insertion Sort

## Tech Stack

* HTML5
* CSS3
* Vanilla JavaScript

## Project Structure

```bash
sorting-visualizer/
│
├── index.html
├── style.css
└── script.js
```

## How to Run

1. Download or clone the project
2. Open `index.html` in your browser

## Visualization Colors

* Blue → Default bars
* Yellow → Comparing bars
* Red → Swapping bars
* Green → Sorted bars

## Purpose of Project

This project was created for learning Data Structures and Algorithms visually in an easy and interactive way.
